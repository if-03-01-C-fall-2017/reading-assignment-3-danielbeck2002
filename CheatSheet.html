CCS Syntax and Selectors

A CSS rule-set consists of a selector and a declaration block.
The selector points to the HTML element you want to style.
The declaration block contains one or more declarations separated by semicolons.
With p { "Content" } you can select all Elemetns on a page.
The id selector selects a specific element " #paraX { "Content" } " would change the style of the first paragraph

CSS Combinators

-The descendant selector matches all elements that are descendants of a specified element.
-The child selector selects all elements that are the immediate children of a specified element.
-The adjacent sibling selector selects all elements that are the adjacent siblings of a specified element.
-The general sibling selector selects all elements that are siblings of a specified element.

CSS Pseudo-classes

CSS Lists

The list-style-type property specifies the type of list item marker. The list-style-image property specifies an image as the list item marker. You can Style lists.


HTML Tables

An HTML table is defined with the <table> tag.
Each table row is defined with the <tr> tag.
A table header is defined with the <th> tag.
By default, table headings are bold and centered. A table data/cell is defined with the <td> tag.

A border is set using the CSS border property. To add a caption to a table, use the <caption> tag


CSS Tables

Width and height of a table are defined by the width and height properties.
For zebra-striped tables, use the nth-child() selector and add a background-color to all even (or odd) table rows.
